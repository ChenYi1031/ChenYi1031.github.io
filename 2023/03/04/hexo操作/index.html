友情链接

  1. 前往你的Hexo根目录
  2. 输入 hexo new page link
  3. 找到 source/link/index.md 这个文件
  4. 编辑这个文件

注意，主题中配置友情链接的位置不是在该文件本身，而是需要你去 创建 source/_data 目录，并且新建文件 link.yml

同时，可以在themes/Butterfly/_config.yml文件中，写上个人博客的信息，方便他人帮忙转载
