什么是 OpenClaw

OpenClaw 是一个开源的 AI 助手框架，可以让你在自己的设备上运行强大的 AI 助手。它支持：

  • 多种聊天平台集成（Telegram、Discord、WhatsApp 等）
  • 丰富的技能扩展系统
  • 自动化任务和定时任务
  • 本地文件操作和 web 搜索
  • 浏览器自动化控制
